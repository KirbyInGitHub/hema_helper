# hema_helper

## 特性

- 目前只支持盒马有运力释放的门店
   - 盒区团购待支持
- 支持抢购和捡漏两种模式
   - 支持设置自定义延迟
- 支持随机选择派送时间
- 支持异常响铃提醒
- 支持下单成功响铃提醒
- 支持自动过淘宝滑块验证
- 支持自动输入支付密码
